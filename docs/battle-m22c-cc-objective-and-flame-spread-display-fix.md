# M22c. CC 객관식 오답 즉시 진행 + 화염확산 표시 보정

## 문제

1. 상대가 CC 상태라 객관식 정답을 제출할 수 없는 상황에서, 행동 가능한 쪽이 오답을 눌러도 타이머를 기다리는 경우가 있었다.
2. 바람+불 화염확산이 발생해도 전투 표시/뱃지가 꽃가루확산으로 잡혔다.

## 원인

- 객관식 오답 즉시 종료 조건이 `stunned`만 확인해 `staggered`, `frozen` 등 다른 퀴즈 차단 CC를 놓쳤다.
- 전투 로그/반응 감지에서 `확산 반응`이라는 넓은 문자열을 꽃가루확산으로 인식해 `화염확산 반응`도 꽃가루확산으로 분류했다.
- `REACTION_FLASH_META`에 `flameSpread` 표시 메타가 없었다.

## 수정 내용

- 상대가 `stunned`, `staggered`, `frozen` 중 하나라도 있으면 객관식 오답 즉시 턴 종료 처리
- `flameSpread` 반응 표시 메타 추가
- `화염확산`은 `flameSpread`로 우선 감지
- 꽃가루확산 감지에서 일반 `확산 반응` 조건 제거
- compact log에서도 화염확산과 꽃가루확산을 분리

## 확인 항목

- `npm run build` 통과
- 상대가 기절/경직/빙결 상태일 때 객관식 오답을 누르면 타이머를 기다리지 않고 즉시 다음 턴으로 진행
- 바람+불 반응 시 뱃지가 화염확산으로 표시
- 화염확산 로그에서 꽃가루확산 문구가 같이 나오지 않음
- 바람+풀 반응은 여전히 꽃가루확산으로 표시
