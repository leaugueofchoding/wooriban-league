# M10.5 follow-up v2. 원소 흔적 유지와 전투 피드백 개선 계획

## 이번 수정

### 원소 흔적 유지

원소 흔적이 묻은 펫이 퀴즈를 맞히고 직접 공격할 때, 자기 몸에 붙어 있던 원소 흔적이 사라지는 문제를 수정합니다.

- 공격자 자신의 원소 흔적은 해당 공격 턴 종료 tick에서 감소시키지 않습니다.
- 원소 흔적은 기본적으로 상대가 후속 원소 공격을 맞아 반응 재료로 소모될 때 사라지는 쪽이 더 직관적입니다.

### 원소 흔적 UI 위치

전투 화면의 원소 흔적 아이콘 트레이를 조금 위로 올립니다.

## 후속 마일스톤 후보: 원소반응 이펙트/사운드

현재 원소반응은 로그와 추가 피해 중심이며, 전용 사운드/화면 이펙트는 별도 구현이 필요합니다.

후속 작업안:

- 감전: 짧은 전기 파열음 + 노란 번개 스파크
- 증발: 물방울 증기 팡 효과 + 파란/주황 혼합 이펙트
- 연소: 작은 폭염/불꽃 확산
- 과부하: 폭발 충격파
- 꽃가루 확산: 꽃가루/바람 소용돌이
- 빙결: 얼음 결정/쩍 소리

## 후속 마일스톤 후보: RPG식 데미지 숫자

상황 로그가 길어지는 문제를 줄이기 위해 캐릭터 위에 데미지 숫자를 띄우는 방식이 적합합니다.

방향:

- 기본공격: 검정/흰 테두리 숫자
- 원소공격: 원소 색 숫자
- 원소반응: 두 원소를 섞은 색 또는 그라데이션 숫자
- 치명타: 더 큰 숫자 + CRITICAL 표시
- 회복: 초록색 + 숫자
- 반응 추가 피해: 별도 작은 숫자 또는 합산 숫자

주의:

- HP 변화와 로그 문자열을 파싱해서 처리하면 깨지기 쉽습니다.
- 장기적으로는 스킬 결과가 `damageEvents` 배열을 반환하도록 구조화하는 편이 안정적입니다.
