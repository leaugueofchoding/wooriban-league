# M22. 바람 원소반응 대기 펫 흔적 확산

## 목표

바람이 포함된 원소반응의 팀전 역할을 직접 피해/CC가 아니라 원소흔적 확산으로 정리한다.

## 규칙

바람이 포함된 원소반응이 발생하면:

- 상대 출전 펫: 기존 원소반응 효과 유지
- 상대 대기 펫: 피해 없음
- 상대 대기 펫: 하드 CC 없음
- 상대 대기 펫 전체: 바람이 아닌 쪽 원소 흔적 부여

## 예시

- 바람 + 불 → 대기 펫들에게 불 흔적
- 바람 + 번개 → 대기 펫들에게 번개 흔적
- 바람 + 물 → 대기 펫들에게 물 흔적
- 바람 + 풀 → 대기 펫들에게 풀 흔적
- 바람 + 얼음 → 대기 펫들에게 얼음 흔적

## 의도

- 바람 펫의 역할을 광역 딜러가 아니라 전장 확산 서포터로 만든다.
- 토네이도 휩쓸기의 직접 광역 피해와 원소반응의 역할이 겹치지 않게 한다.
- 대기 펫에게 즉시 피해/화상/경직을 주지 않아 불쾌감을 줄인다.
- 대신 교체 후 추가 원소반응 가능성을 만들어 팀전 전략성을 높인다.

## 확인 항목

- `npm run build` 통과
- 불 흔적이 있는 상대에게 바람 스킬 사용 시 대기 펫에 불 흔적 부여
- 바람 흔적이 있는 상대에게 불 스킬 사용 시 대기 펫에 불 흔적 부여
- 대기 펫 HP는 감소하지 않음
- 대기 펫에게 화상/경직/기절은 바로 생기지 않음
