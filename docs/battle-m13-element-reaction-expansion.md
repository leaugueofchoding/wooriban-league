# M13. 원소반응 확장 1차

## 추가 반응

| 조합 | 이름 | 효과 |
| --- | --- | --- |
| 물 + 풀 | 만개 | 속박 1턴 |
| 물 + 바람 | 소용돌이 | 혼란 1턴 |
| 번개 + 바람 | 폭풍 | 경직 1턴 |
| 번개 + 얼음 | 초전도 | 욱신/약화 1턴 |
| 번개 + 풀 | 과성장 | 공격자 HP 8% 회복 |
| 풀 + 얼음 | 서리덩굴 | 욱신/약화 1턴 |

## 설계 원칙

- 모든 반응을 기절/폭딜로 만들지 않는다.
- 속박, 혼란, 경직, 약화, 회복으로 역할을 분산한다.
- 강한 하드 CC는 빙결 정도로 제한한다.
- 과성장은 방어자에게 피해/CC를 주기보다 공격자 회복으로 차별화한다.

## 확인 항목

- 물+풀: 만개 플래시/사운드, 속박 적용
- 물+바람: 소용돌이 플래시/사운드, 혼란 적용
- 번개+바람: 폭풍 플래시/사운드, 경직 적용
- 번개+얼음: 초전도 플래시/사운드, 약화 적용
- 번개+풀: 과성장 플래시/사운드, 공격자 HP 회복
- 풀+얼음: 서리덩굴 플래시/사운드, 약화 적용


## M13 v2 데미지 밸런스

원소반응을 유도할 이유가 생기도록 전체 반응 피해를 상향한다.

원칙:

- CC 없는 반응은 피해를 강하게 둔다.
- 강한 CC가 붙은 반응은 피해를 낮게 둔다.
- 속박/혼란/경직/약화처럼 전투 흐름을 바꾸는 반응은 중간 이하 피해로 둔다.
- 한 번에 게임이 끝나는 수준의 폭딜은 피한다.

조정 예시:

- 증발, 과부하, 감전: 피해 중심 반응으로 강화
- 빙결: 강한 CC이므로 피해는 낮게 유지
- 만개/소용돌이/폭풍: CC가 있으므로 피해는 중간 이하
- 초전도/서리덩굴: 약화 디버프 + 중간 피해
- 과성장: 소량 피해 + 공격자 회복
- 뇌우: 경직이 아니라 기절로 조정
