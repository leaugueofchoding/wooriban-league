# M21b. 토네이도 광역 피해 저장 보정 + 로그 말투 정리

## 목표

M21 적용 뒤 토네이도 휩쓸기의 대기 펫 피해가 실제 전투 상태에 남지 않는 문제를 보정한다.  
또 전투 로그에서 불필요한 안내 문구를 제거하고, 주요 로그 말투를 반말로 통일한다.

## 수정 내용

- `BattlePage.jsx`
  - 토네이도 휩쓸기 사용 후 대기 펫 피해 fallback 적용
  - 이미 `petData.js`에서 대기 펫 피해가 적용된 경우에는 중복 적용하지 않음
  - 대기 펫 피해는 출전 펫 최종 피해의 35%
  - 대기 펫은 HP 1 아래로 내려가지 않음
  - 표시 로그에서 “이번 턴에는 더 이상 교체할 수 없습니다.” 제거
  - 표시 로그의 주요 존댓말 표현을 반말로 정리
  - CC 방어 로그 문구를 반말로 정리

- `petData.js`
  - 토네이도/기절 관련 일부 로그 문구를 반말로 정리

## 확인 항목

- `npm run build` 통과
- 토네이도 휩쓸기 사용 시 상대 대기 펫 HP 감소 확인
- 같은 대기 펫 피해가 중복으로 들어가지 않는지 확인
- 교체 로그에서 “이번 턴에는 더 이상 교체할 수 없습니다.” 문구가 보이지 않는지 확인
- 기절/경직/혼란 관련 로그가 반말로 표시되는지 확인
