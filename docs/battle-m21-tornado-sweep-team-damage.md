# M21. 토네이도 휩쓸기 광역 피해 적용

## 작업 내용

- 토네이도 휩쓸기를 실제 광역 피해 스킬로 변경
- 출전 펫은 기존 피해 유지
- 대기 펫은 출전 펫 최종 피해의 35% 피해
- 대기 펫은 광역 피해로 쓰러지지 않고 HP 1 보호
- 경직은 출전 펫에게만 적용

## M21b 보정

- 토네이도 광역 피해가 저장되지 않던 문제를 BattlePage resolution 단계에서 보정
- 이미 petData에서 대기 펫 피해가 적용된 경우 중복 적용하지 않음
- 전투 로그 말투를 반말 중심으로 정리
- 불필요한 교체 안내 문구 제거

## 확인

- npm run build 통과
- 토네이도 휩쓸기 출전 펫 피해 확인
- 토네이도 휩쓸기 대기 펫 피해 확인
- 대기 펫 HP 1 보호 확인
