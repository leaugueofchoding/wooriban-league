# M25c-2. 펫페이지 스킬 툴팁 잘림 수정 v2

## 문제

스킬 카드 내부 CSS 툴팁이 부모 영역의 `overflow`에 잘려 보였다.

## 변경 내용

- 카드 내부 pseudo 툴팁 대신 fixed 툴팁 레이어 사용
- 부모 박스, 스크롤 영역, 파란 테두리 영역에 잘리지 않도록 수정
- `pendingSkillId` 유무와 관계없이 state 삽입 가능하도록 보정
- 기존 카드 안 설명은 2줄 말줄임 유지

## 확인

- npm run build 통과
- 펫페이지 > 스킬 관리
- 장착 슬롯/보유 스킬 카드 호버 시 툴팁이 잘리지 않고 표시
