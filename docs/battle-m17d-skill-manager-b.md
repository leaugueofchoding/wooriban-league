# M17D. 스킬 관리 UI B안

## 목표

A안의 장착 슬롯 + 보유 스킬 리스트 구조가 세로로 길어지는 문제를 줄이기 위해, 더 짧은 2열 미니 카드 구조로 변경한다.

## 변경 방향

- 장착 슬롯과 보유 스킬을 모두 2열 미니 카드로 표시한다.
- 스킬 설명은 최대 2줄까지만 표시한다.
- 카드 패딩, 글자 크기, 카드 높이를 줄인다.
- 비법 노트와 저장하기 버튼은 유지한다.

## 장착 흐름

1. 교체할 슬롯을 먼저 선택한다.
2. 보유 스킬을 클릭하면 해당 슬롯에 장착된다.
3. 빈 슬롯이 있으면 보유 스킬 클릭만으로 바로 장착된다.
4. 슬롯이 꽉 찼는데 슬롯을 고르지 않으면 안내 문구를 띄운다.
5. 고유 스킬은 교체할 수 없다.

## 확인 항목

- `/pet` 스킬 관리 탭이 A안보다 세로로 짧아졌는지
- 장착 슬롯이 2열 미니 카드로 보이는지
- 보유 스킬도 2열 미니 카드로 보이는지
- 설명이 2줄에서 잘리는지
- 빈 슬롯 바로 장착이 되는지
- 선택 슬롯 교체가 되는지
- 고유 스킬 교체가 막히는지
- 저장하기가 정상 작동하는지
- `npm run build` 통과
