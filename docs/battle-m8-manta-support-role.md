# M8. 가오리 계열 1차 서포터화

M8은 가오리 계열을 한 방 폭딜러가 아니라 물결표식과 물 흔적을 쌓는 서포터로 정리하는 단계입니다.

## 마이그레이션 판단

기존 학생들이 이미 보유한 가오리는 별도 마이그레이션이 필요하지 않습니다.

- 스킬 ID는 유지합니다.
- 기존 가오리의 장착 스킬 구조를 바꾸지 않습니다.
- `SKILLS.WAVE_MARK`, `SKILLS.BLOSSOM_CURRENT`, `SKILLS.ARA_BLOOM`의 현재 로직과 계수만 조정합니다.
- 따라서 기존 가오리도 새 계수와 새 메커니즘을 자동으로 사용합니다.

## 변경 방향

| 스킬 | 기존 역할 | M8 방향 |
| --- | --- | --- |
| 물방울 낙인 | 저비용 표식 + 2스택 기절 | 저피해 표식 누적기 + 3스택 기절 |
| 벚꽃해류 | 피해 + 표식 + 자기 회복 | 피해 하향, 표식/회복 보조 강화 |
| 아라만개 | 3표식 기준 큰 폭딜 | 표식 회수형 정리기로 완화 |

## 주요 변경

- 물결표식 기절 조건을 2개에서 3개로 완화
- 아라만개 표식 배율을 `1.4 / 3.0 / 6.5`에서 `1.25 / 2.2 / 3.8`로 하향
- 물방울 낙인, 벚꽃해류, 아라만개의 직접 피해 계수 하향
- 벚꽃해류의 자기 회복은 표식 수당 5%로 유지하여 서포터 체감을 살림

## 보류: 팀 전체 지속 회복

벚꽃해류를 대기열 포함 팀 전체 지속 회복으로 바꾸는 아이디어는 좋지만 M8에서는 넣지 않습니다. 이 기능은 active pet만이 아니라 team 배열 전체에 지속 상태를 저장하고, 대기 펫이 다시 등장해도 남은 회복 턴을 유지해야 합니다.

후속 단계 후보:

- M8.5 또는 M12: `blossomRegen` 팀 버프 구조 추가
- 대기열 포함 팀 전체 회복 처리
- 교체 후에도 회복 버프 유지
- 로그 압축
- UI 표시 방식 정리

예상 설계:

| 상대 표식 | 팀 전체 회복안 |
| --- | --- |
| 1개 | 3턴간 턴 종료 시 팀 전체 최대 HP 5% 회복 |
| 2개 | 3턴간 턴 종료 시 팀 전체 최대 HP 10% 회복 |
| 3개 | 3턴간 턴 종료 시 팀 전체 최대 HP 15% 회복 |

## 검증 포인트

- 기존 가오리 보유 데이터가 깨지지 않는지
- 물방울 낙인 사용 시 표식이 정상 증가하는지
- 표식 3개 도달 시 기절이 정상 적용되는지
- 벚꽃해류 사용 시 표식과 회복이 정상 동작하는지
- 아라만개 사용 시 표식 수에 따라 피해가 증가하고, 명중 후 표식이 사라지는지
