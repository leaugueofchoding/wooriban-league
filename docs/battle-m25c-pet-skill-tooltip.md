# M25c. 펫페이지 스킬 교체 툴팁 v2

## 문제

펫페이지의 스킬 교체 화면에서 스킬 카드가 작아 긴 설명이 2줄로 잘려 보였다.

## 변경 내용

- 스킬 교체 화면의 장착 슬롯/보유 스킬 카드에 마우스를 올리면 전체 설명 툴팁 표시
- 기본 브라우저 `title`도 함께 제공
- 툴팁에는 다음 정보 표시
  - 스킬명
  - 속성
  - 현재 레벨 기준 SP
  - 위력
  - 전체 설명
- 키보드 포커스 시에도 툴팁 표시 가능
- 기존 카드 안 설명은 2줄 말줄임 유지

## 확인 항목

- `npm run build` 통과
- 펫페이지 > 스킬 관리 열기
- 장착 슬롯의 스킬 카드에 마우스 올리면 전체 설명 표시
- 보유 스킬 카드에 마우스 올리면 전체 설명 표시
