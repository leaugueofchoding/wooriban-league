# M24d. 뇌우 대기열 기절 표시 개선

## 목표

뇌우로 대기 펫이 기절했을 때 학생이 상황을 더 명확히 이해할 수 있도록 로그와 상태 표시 흐름을 개선한다.

## 변경 내용

### 1. 대기열 기절 로그 문구 개선

기존 compact log에서는 기절 문구가 일반적으로 `상대가 움직이지 못한다!`로 표시될 수 있었다.

변경 후:

- 뇌우로 대기 펫이 기절한 경우 `OOO이 기절했습니다.`로 표시
- 여러 마리가 기절한 경우 기존 로그의 이름 요약을 활용

### 2. 기절 대기 펫 수동교체 표시 개선

기절한 대기 펫이 수동교체로 출전할 때:

- 교체 직후에는 기절 상태를 바로 지우지 않음
- 잠깐 동안 기절 CC 테두리와 상태 카드가 표시됨
- 교체 연출이 끝나고 다음 문제턴으로 넘어갈 때 기절 상태를 제거
- 기본공격은 나가지 않음

## 확인 항목

- `npm run build` 통과
- 뇌우로 대기 펫 기절 시 로그가 `OOO이 기절했습니다.` 형태로 표시
- 기절한 대기 펫 수동교체 시 기절 테두리/상태 카드가 잠깐 표시
- 다음 문제턴 시작 후 기절 카드가 사라짐
- 타이머 만료 기본공격이 나가지 않음
