# M15. 도감형 펫 도감 레이아웃 개편

## 목표

펫 도감을 전투 중 도움말이 아니라, 펫/진화/속성/원소반응/상성을 확인하는 전용 화면으로 정리한다.

## 적용 내용

- 기존 세로형 계열 나열 도감을 도감형 좌우 레이아웃으로 변경
- 왼쪽에는 작은 펫 아이콘 그리드 표시
- 오른쪽에는 선택한 펫의 상세 카드 표시
- 도감 번호 형태의 No. 표시 추가
- 발견하지 못한 펫은 실루엣으로 표시
- 우리 반에서 발견된 펫은 채색으로 표시
- 내가 보유한 펫은 초록 계열 카드/테두리로 표시
- 선택한 펫은 붉은 테두리로 표시
- 펫을 클릭하면 오른쪽 상세 카드가 해당 펫으로 변경
- 상세 카드에 속성, 진화 단계, 조건, 전투 역할, 스킬, 상성, 발견 친구 표시
- 원소반응·상성 노트를 상세 카드 하단으로 이동

## 판정 기준

- 전투 화면에는 긴 도움말을 넣지 않는다.
- 도감에서 규칙을 읽고, 전투에서는 짧은 로그만 본다.
- 펫 도감은 앞으로 신규 펫/얼음 펫이 추가되어도 그리드에 자연스럽게 늘어나는 구조로 둔다.

## 확인 항목

- /pet-dex에서 도감형 그리드가 보이는지
- 미발견 펫이 실루엣으로 보이는지
- 발견된 펫이 채색으로 보이는지
- 내가 보유한 펫이 초록 카드/테두리로 구분되는지
- 펫 클릭 시 오른쪽 상세 카드가 바뀌는지
- 원소반응·상성 노트가 오른쪽 카드 하단에 있는지
- 태블릿/모바일에서 오른쪽 카드가 아래로 내려오며 깨지지 않는지
