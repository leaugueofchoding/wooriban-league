# M17E v2. 펫 도감 가로폭 정렬

## 목표

- 펫 도감 본문 폭을 헤더 폭과 맞춘다.
- 왼쪽 기준은 `펫 도감` 제목 시작점이다.
- 오른쪽 기준은 `내 수집 현황` 카드 오른쪽 끝이다.
- 태블릿 가로 환경에서 도감 그리드와 상세 카드가 나란히 보이도록 유지한다.

## 변경 내용

- `.dex-shell`을 `width: min(1120px, calc(100vw - 1.3rem))` 기반으로 넓힌다.
- `.pokedex-layout`을 전체 폭 사용 구조로 바꾼다.
- 태블릿 구간에서도 `.dex-shell`을 960px로 강제 축소하지 않는다.
- 860px 이하에서는 1열 전환을 유지한다.

## 확인 항목

- `/pet-dex`에서 도감 본문 왼쪽 끝이 제목 시작점과 맞는지 확인
- 도감 본문 오른쪽 끝이 내 수집 현황 카드 오른쪽 끝과 맞는지 확인
- 1024px 안팎 태블릿 가로 화면에서 2열이 유지되는지 확인
- 860px 이하에서는 1열로 자연스럽게 내려가는지 확인
- `npm run build` 통과
