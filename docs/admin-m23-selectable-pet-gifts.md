# M23. 관리자 선물 시스템 개편

## 목표

대규모 전투 시스템 업데이트 기념 선물을 쉽게 지급할 수 있도록 관리자 선물 기능을 개편한다.

## 변경 내용

- 관리자 선물 탭에서 학생을 직접 선택할 수 있음
- 전체 선택 / 전체 해제 가능
- 학생 이름 검색 가능
- 지급할 펫 아이템을 직접 선택할 수 있음
- 아이템 수량 선택 가능
- 여러 아이템을 한 번에 지급 가능
- 학생에게 표시될 선물 제목과 메시지 직접 입력 가능
- 학생 접속 시 기존 선물 모달에 지급 내용 표시

## 기본 선물 문구

제목:

`🎁 업데이트 기념 선물이 도착했어!`

메시지:

`대규모 배틀 업데이트 기념 선물이야! 새로운 속성 흔적 반응을 재미있게 즐겨 봐!`

## 확인 항목

- `npm run build` 통과
- 관리자 선물 탭 진입 확인
- 학생 선택 가능
- 아이템 추가/삭제 가능
- 선택 학생에게만 지급 가능
- 학생 계정에서 선물 모달 표시
- 펫 가방에 수량 반영
