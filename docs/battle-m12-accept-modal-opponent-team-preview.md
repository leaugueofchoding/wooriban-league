# M12. 대전 수락창 상대 펫/레벨 표시

## 목표

대전 수락창 상단에서 상대방이 어떤 펫으로 신청했는지 바로 확인할 수 있게 한다.

## 변경 내용

- 수락창 상단에 상대 출전 펫 카드 목록을 표시한다.
- 1:1이면 상대 펫 1마리만 표시한다.
- 2:2/3:3이면 상대 팀의 모든 출전 펫을 표시한다.
- 각 카드에 펫 이미지, 이름, 레벨, HP를 표시한다.
- 기존 수락/거절 로직과 팀 선택 로직은 변경하지 않는다.

## 확인 항목

- 1:1 신청 수락창에서 상대 펫 1마리와 레벨이 보이는지
- 2:2/3:3 신청 수락창에서 상대 팀 전체가 보이는지
- 태블릿/모바일 화면에서 수락/거절 버튼까지 스크롤 가능한지
- 수락/거절 동작이 기존처럼 정상 작동하는지


## v12c 레이아웃 보정

- 수락창 헤더를 "{도전자 이름}님의 도전장이 도착했습니다!" 형태로 보정했다.
- 카드 내부를 가로형에서 세로형으로 바꿨다.
- 펫 이름 잘림을 줄이기 위해 이미지 아래에 이름/레벨/HP를 줄바꿈하여 쌓았다.
- "000의 도전", "0vs0 대전 신청" 문구는 제거했다.
- 펫 번호 표시는 제거했다.
