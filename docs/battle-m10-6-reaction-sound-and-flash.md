# M10.6. 원소반응 사운드와 플래시 이펙트

M10.6은 원소반응이 실제로 터졌다는 느낌을 강화하는 시각/청각 피드백 단계입니다.

## 변경

- 원소반응 로그를 감지해 전투장 중앙에 반응 플래시 표시
- 반응별 아이콘/색상 조합 적용
- Web Audio API 기반 전용 원소반응 사운드 추가
- 외부 음원 파일 추가 없음

## 반응별 연출

| 반응 | 연출 방향 |
| --- | --- |
| 감전 | 물방울 + 전기 스파크 |
| 증발 | 물방울 + 증기 폭발 |
| 연소 | 풀 질감 + 불 번짐 |
| 과부하 | 전기 충전 후 폭발 |
| 꽃가루 확산 | 바람 + 반짝이는 꽃가루 |
| 빙결 | 얼음 결정음 |

## 보류

RPG식 데미지 숫자는 M10.7로 분리합니다.

이유:

- 로그 문자열을 파싱해 숫자를 띄우면 깨지기 쉽습니다.
- 안정적으로 만들려면 스킬 결과가 `damageEvents` 배열을 반환하도록 구조화하는 편이 좋습니다.
- 원소공격/반응/치명타/회복을 모두 같은 이벤트 시스템으로 처리해야 합니다.
