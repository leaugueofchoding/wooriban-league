# M24. 콰릉숭 뇌우 대기열 기절

## 목표

콰릉숭의 최종 스킬 `뇌우`가 팀 배틀에서 더 확실한 광역 압박 역할을 하도록 개선한다.

## 변경 내용

- 뇌우 기절 확률을 25%로 통일
- 스킬 설명도 25%로 수정
- 뇌우 사용 시 상대 대기 펫 각각에게 25% 기절 판정
- 랜덤 1마리만 선택하지 않음
- 대기 펫에게 직접 피해는 주지 않음
- 쓰러진 대기 펫은 제외
- 이미 기절/경직/빙결 중인 대기 펫은 중복 기절 제외
- 기절한 대기 펫 이름을 전투 로그에 표시

## 의도

`토네이도 휩쓸기`가 대기 펫 HP를 압박하는 광역기라면,
`뇌우`는 대기 펫의 다음 출전 타이밍을 흔드는 광역 CC 압박기로 분리한다.

## 확인 항목

- `npm run build` 통과
- 콰릉숭이 뇌우 사용 시 출전 펫에게 기존 피해가 들어감
- 출전 펫 기절 확률이 25%로 동작
- 상대 대기 펫 각각에게 25% 기절 판정이 적용됨
- 대기 펫 HP는 감소하지 않음
- 기절한 대기 펫 이름이 로그에 표시됨
