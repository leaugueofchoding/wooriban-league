# M9. 기존 타입 상성 삭제와 물결표식 연계

M9에서는 기존 단순 타입 상성 배율을 실제 계산에서 제거하고, 강한 결과를 원소 흔적/원소반응과 표식 연계로 이동합니다.

## 타입 상성 삭제

기존 `ELEMENT_CHART` 기반 상성 배율은 삭제합니다.

- 기존: 스킬 속성이 상대 속성에 유리하면 피해 1.3배
- 변경: 속성만으로는 피해 배율을 직접 올리지 않음
- 앞으로 강한 결과는 원소 흔적과 원소반응에서 발생

`calculateDamage`는 기존 호출 호환성을 위해 속성 인자를 받지만, 더 이상 단순 상성 배율에는 사용하지 않습니다.

## 물결표식과 기본공격

물결표식은 가오리만의 후속타가 아니라 팀 전체가 활용할 수 있는 약점 표식으로 정리합니다.

| 물결표식 | 기본공격 피해 |
| --- | --- |
| 0개 | 1.0배 |
| 1개 | 1.2배 |
| 2개 | 1.45배 |
| 3개 | 1.75배 |

이 효과는 가오리뿐 아니라 현재 필드에 나온 팀원의 기본공격에도 적용됩니다.

## 물결표식과 물 원소 흔적

물결표식과 물 원소 흔적은 서로 다른 상태입니다.

- 물결표식: 가오리의 표식/약점 시스템
- 물 원소 흔적: 원소반응 엔진의 흔적 시스템

다만 물 원소 흔적이 원소반응으로 소모되면, 물결표식도 함께 사라집니다.

예시:

1. 가오리가 물결표식을 쌓음
2. 물 원소 흔적이 남아 있음
3. 번개 공격으로 감전 반응이 발생하며 water trace가 소모됨
4. 남아 있던 물결표식도 함께 사라짐

## 배포 판단

전체 원소반응 개편 작업을 마친 뒤 배포할 예정이므로, 이번 단계에서는 legacy flag 분기를 남기지 않고 단순 상성 계산을 바로 제거합니다.
