# M10.7 v5. 바람의 칼날 피격감과 꽃가루 확산 감지 수정

## 수정 1. 바람의 칼날 피격감

데미지 숫자는 Firestore HP 변화 감지 후 표시된다.

기존에는 스킬 애니메이션 쪽의 피격 진동과 HP 변화 감지 쪽의 데미지 숫자가 서로 어긋나서,
특히 바람의 칼날 연타가 밋밋하게 느껴질 수 있었다.

수정:

- HP 변화로 데미지 숫자를 만드는 바로 그 시점에 피격 진동도 다시 발생시킨다.
- 바람의 칼날처럼 숫자가 여러 번 뜨는 스킬은 진동도 짧게 여러 번 울린다.
- 바람의 칼날 resolution 시작 타이밍을 더 앞당겼다.

## 수정 2. 꽃가루 확산 중앙 플래시/사운드

풀+바람 반응은 실제로는 발생해 원소 흔적이 소모되었지만,
로그 문구가 `꽃가루 확산`이 아니라 `확산 반응`으로 들어와 중앙 플래시/사운드 감지에서 빠질 수 있었다.

수정:

- `꽃가루 확산`
- `확산 반응`
- `꽃가루가 퍼져`
- `바람을 타고 ... 꽃가루`

위 문구를 모두 `pollenSpread` 반응으로 감지한다.
