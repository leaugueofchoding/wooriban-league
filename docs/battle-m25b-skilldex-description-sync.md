# M25b. 스킬도감 설명 일치화 및 포롱이 밸런스 v3

## 목표

스킬도감에 표시되는 설명과 실제 전투 효과가 어긋나지 않도록 `petData.js`의 스킬 설명과 미리보기 상태 문구를 정리한다.

## 포롱이 계열 밸런스 v3

- 벚꽃해류: 피해 계수 1.10, 표식당 최대 HP 7% 회복, 대기 펫 회복량 50%
- 아라만개: 표식 1/2/3개 피해 배율 1.4/2.5/4.5

## 설명 일치화

- 씨뿌리기: 준 피해 50% 회복
- 덩굴 채찍: 1턴 공격/방어 30% 감소
- 스텔라 블라스트: 22% 화상
- 따끔할퀴기: 35% 경직
- 업화: 체력 20% 이하 1.6배 + 45% 역전 치명타, 화상 대상 1.2배
- 반격태세: 다음 공격 25% 반사
- 부들화살: 방어/교체/도망 봉쇄
- 뇌우: 출전/대기 각각 25% 기절

## 확인 항목

- `npm run build` 통과
- 펫 도감 스킬 설명과 실제 수치 일치
- 스킬 미리보기 카드의 상태 문구와 실제 효과 일치
