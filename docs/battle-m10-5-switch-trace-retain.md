# M10.5 follow-up v3. 교체 중 원소 흔적 유지

## 문제

원소 흔적이 묻은 펫을 교체할 때 원소 흔적이 간헐적으로 사라지는 현상이 있었다.

원인 후보:

- 원소 흔적은 남은 턴 수를 가진 상태다.
- 수동 교체도 한 턴 종료로 처리된다.
- 이때 교체하는 펫의 원소 흔적도 tick되어, 남은 턴이 1이면 교체 순간 사라질 수 있다.
- 그래서 규칙 없이 사라지는 것처럼 보였다.

## 수정

- 공격자가 직접 공격할 때에 이어, 수동 교체와 아이템 사용에서도 행동자 자신의 원소 흔적은 tick하지 않는다.
- 원소 흔적은 상대가 후속 원소 공격을 맞아 반응 재료로 소모될 때 사라지는 것을 기본 규칙으로 둔다.

## UI

원소 흔적 아이콘 트레이를 전투 화면에서 더 위로 올렸다.
