# M24b. 기절 대기 펫 수동교체 처리

## 문제

뇌우로 기절한 대기 펫을 수동교체로 꺼냈을 때,
기절 상태인데도 액션 타이머가 끝나면 기본공격이 나가는 문제가 있었다.

## 변경 내용

- 기절/경직/빙결 상태인 대기 펫을 수동교체로 꺼낼 수는 있음
- 단, 출전 직후 공격 기회는 스킵
- 로그에 행동불가 이유 표시
- 잠깐 교체 연출을 보여준 뒤 다음 문제턴으로 이동
- 타이머 만료 후 기본공격으로 처리되는 흐름 차단
- 해당 행동불가 상태는 1회 소모 처리

## 확인 항목

- `npm run build` 통과
- 뇌우로 대기 펫 기절 발생
- 기절한 대기 펫을 수동교체로 출전
- 교체된 펫이 잠깐 등장함
- 로그에 기절 상태라 움직일 수 없다는 안내가 뜸
- 기본공격이 나가지 않음
- 바로 액션 타이머를 기다리지 않고 다음 문제턴으로 넘어감
