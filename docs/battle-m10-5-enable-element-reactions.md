# M10.5. 원소반응 실제 활성화

M10.5에서는 원소반응 시스템을 실제 전투에 켭니다.

## 변경

- `FEATURE_FLAGS.ELEMENT_REACTION_ENABLED`를 `true`로 전환
- M9에서 기존 단순 타입 상성은 이미 제거됨
- `LEGACY_TYPE_CHART_ENABLED`는 더 이상 사용하지 않음
- 테스트 스크립트의 stale legacy flag 필드 정리

## 같은 턴 흔적 감소 방지

원소 흔적은 공격/스킬 처리 직후 붙습니다. 기존 흐름에서는 같은 resolution 안에서 턴 종료 처리가 이어지기 때문에, 새로 붙은 흔적이 그 턴에 바로 1 감소할 수 있습니다.

이를 막기 위해 이번 행동으로 새로 붙은 흔적에는 `elementTraceJustApplied` marker를 남기고, 첫 턴 종료 tick에서 marker만 제거합니다.

## 확인해야 할 화면 동작

- 원소 스킬 적중 시 작은 원소 흔적 UI가 보이는지
- 다른 원소 스킬로 반응이 터지는지
- 반응 로그가 뜨는지
- 반응 추가 피해가 적용되는지
- 물 원소 흔적이 소모될 때 물결표식도 함께 사라지는지
- 기존 큰 상태이상 카드와 원소 흔적 UI가 섞이지 않는지

## 1차 확인 조합

| 선행 흔적 | 후속 공격 | 반응 |
| --- | --- | --- |
| 물 | 번개 | 감전 |
| 물 | 불 | 증발 |
| 불 | 풀 | 연소 |
| 불 | 번개 | 과부하 |
| 풀 | 바람 | 꽃가루 확산 |
| 물 | 얼음 | 빙결 |

얼음 계열은 아직 실제 펫이 없다면 코드 테스트로만 확인합니다.
