# M10.7 v2. 데미지 숫자 UX 조정

## 변경

- 데미지 숫자 크기 확대
- 치명타는 훨씬 큰 숫자와 CRITICAL 라벨로 표시
- 피해 숫자의 `-` 제거
- 회복만 `+` 표시
- 바람 데미지는 회색 글자 + 검은 윤곽선으로 변경
- 회복 로그가 섞인 공격에서 상대 위에 초록 회복 숫자가 뜨는 문제 수정
- 연타 컨셉 스킬은 총 피해량을 나누어 여러 숫자로 순차 표시
- 상황 로그는 화면 표시 단계에서 피해 숫자/계산식성 문구를 줄이고 이유 중심으로 축약

## 연타 표시 대상

- 바람의 칼날: 3타
- 용의 발톱: 2타
- 재빠른 교란: 2타
- 오의필살/비전 오의: 3타

## 메모

현재는 HP 변화량과 로그 텍스트 기반의 1차 구현입니다.
나중에 `damageEvents` 구조를 만들면 스킬별 타수, 치명타, 원소반응 추가 피해를 더 정확하게 분리할 수 있습니다.
