# M10.7 v4. 상황로그 간결화

## 목표

데미지 숫자가 전투장 위에 표시되기 시작했으므로, 상황로그는 더 이상 피해량과 계산 결과를 길게 보여줄 필요가 없다.

상황로그는 다음 정보만 간결하게 보여준다.

- 누가 어떤 스킬을 썼는지
- 원소반응이 발생했는지
- 반응 없는 턴의 핵심 상태 변화

## 표시 예시

```
'꽃잎치'의 덩굴 채찍! 🌿 💢
연소 반응! 🔥🌿 풀에 불이 옮겨붙었다!
```

## 원칙

- 피해량 숫자는 상황로그에서 제거
- 계산식성 문구 제거
- 종류가 다른 로그는 줄바꿈
- 말투는 반말/게임 로그체로 통일
- 원소반응이 발생한 턴에는 흔적 소모/부가 설명을 숨겨 로그를 짧게 유지
- 원소반응이 없는 턴에는 원소 흔적/상태이상을 한 줄로 표시
