# M17B. 펫 페이지/펫센터 도감 스타일 보정

## 보정 이유

M17 초안은 모든 펫 관련 영역을 지나치게 그리드화하여 `/pet` 화면에서 보유 펫 목록과 선택 펫 상세 카드의 위치감이 어색했다.

이번 보정은 도감 그리드의 구조 전체가 아니라 다음 디자인 언어만 가져온다.

- 파란 프레임
- 작은 픽셀 카드
- 압축된 정보 밀도
- 선택 상태의 굵은 테두리
- 태블릿 가로 화면에서 한눈에 들어오는 2열 구성

## /pet 최종 방향

- 왼쪽: 보유 펫 목록
- 오른쪽: 선택한 펫 상세 카드
- 보유 펫 목록은 도감형 작은 카드 그리드를 유지
- 선택 펫 상세는 기존 큰 흰 카드가 아니라 도감형 상세 패널로 재정리
- 모바일에서는 보유 펫 목록이 위, 상세 카드가 아래로 내려간다.

## /pet-center 최종 방향

- 상점과 치료소는 카드형 레이아웃 유지
- 도감형 파란 프레임과 압축 카드 톤만 적용
- 태블릿에서 세로로 너무 길어지지 않도록 내부 스크롤 높이를 제한

## 확인 항목

- `/pet`에서 보유 펫 목록이 왼쪽에 위치하는지
- `/pet`에서 선택 펫 상세가 오른쪽에 위치하는지
- 선택 펫 카드가 기존 흰 카드 느낌이 아니라 도감형 상세 패널처럼 보이는지
- 태블릿 가로 폭에서 화면이 지나치게 길어지지 않는지
- 모바일에서는 보유 목록 → 상세 순서로 자연스럽게 내려오는지
- `/pet-center` 치료소/상점이 태블릿에서 보기 편한 카드 밀도를 유지하는지
- `npm run build` 통과
